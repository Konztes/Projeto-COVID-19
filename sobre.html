O que é o nosso projeto?

O "projeto API COVID-19" foi criado para passar as informações mais
relevantes a respeito da pandemia da COVID-19.


Nosso objetivo

Nosso objetivo é transmitir os dados sobre a COVID-19 de forma simples e detalhada,
garantindo que todos que procuram estar informados, recebam o assunto sempre atualizado, e
buscamos sempre manter vocês atentos com os ultimos acontecimentos.


Tecnologias utilizadas neste projeto

--> Nosso objetivo foi desenvolvido graças a capacidade de aplicação da linguagem de programação HTML,
que permite a criação do corpo e design da página.

--> JavaScript, que nos permite configurar as funções de cada elemento da página, e transferir
os dados necessarios para os gráficos.

--> CSS, que nos ajuda a dar vida à nossa página, aprimorando o design com cores, textos personalizados, formatos variaveis etc.


Créditos da origem dos dados

Os dados utilizados na formação deste projeto foram retirados do site "Covid19 Brazil API".

Link dos dados: https://covid19-brazil-api-docs.vercel.app/
